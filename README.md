## 📚 Contenido

### 🌿 Guía Cannabis (Interior + Exterior)

#### Modo Interior
- **Compra** — Kit completo (armario, LED, extractor, sustrato, nutrientes)
- **Semillas** — Qué elegir, variedades recomendadas
- **Setup** — Montaje del armario, presión negativa, verificación
- **Riego Automático** — Sistema de goteo, temporizador
- **Germinación** — Papel húmedo, 5-7 días
- **Vegetativa** — Días 7-35, riego, luz, nutrientes, pH
- **Poda & LST** — Técnicas seguras para autoflorecientes, ScrOG
- **Floración** — 35-80 días, cambio a 12/12h (fotoperiodo)
- **Plagas Interior** — Fungus gnats, spider mites, trips
- **Cosecha** — Tricomas, drying, curing (2-8 semanas)

#### Modo Exterior
- **Compra** — Lista optimizada para bosque
- **Semillas** — Fotoperiodo vs auto, variedades mediterráneas
- **Ubicación** — Criteria, camuflaje, seguridad digital
- **Riego** — Frecuencia por mes, sistemas automáticos
- **Fases** — Calendario mes a mes (marzo a octubre)
- **Plagas** — Orugas, caracoles, Botrytis (crítico sept-oct), oídio
- **Cosecha** — Trichomas, hermafroditas, recuperación

#### General (Aplica a ambos modos)
- **Timeline** — Visualización temporal interactiva
- **Mis Semillas** — Perfil de tus variedades actuales
- **Diagnóstico** — Deficiencias de N/P/K/Mg/Ca/Fe, toxicidad
- **Extracciones** — Hash seco, hash agua, mantequilla cannábica
- **FAQ** — 20+ preguntas comunes
- **Glosario** — 50+ términos explicados

### 🍄 Guía Setas Psilocybe cubensis

#### Modo Kit (Lo más fácil — 2-3 semanas)
- **Compra** — Pan de micelio, SGFC, perlita, spray, termómetro
- **Preparación** — Montar cámara, condiciones (20-25°C, 90-95% HR)
- **Incubación** — Rutina diaria, identificar colonización
- **Fructificación** — Primordios, nebulización, FAE
- **Cosecha** — Técnica giro-tirón, velo, cold shock
- **Secado** — Presecado, cracker dry, conservación

#### Modo Manual (Desde cero — 4-8 semanas, máximo control)
- **Compra** — Olla a presión, frascos, granos, SAB, jeringa esporas
- **Esterilización** — Preparación grano, tiempos, enfriamiento
- **Inoculación** — Protocolo aséptico, SAB, técnica aguja
- **Incubación** — Colonización (7-21 días), agitar, detectar contaminación
- **Fructificación** — Bulk tek, monotub, condiciones
- **Cosecha** — Múltiples flushes (3-6), rehydration
- **Secado** — Igual que kit, largo plazo

#### General
- **Glosario** — Micelio, primordio, flush, FAE, SGFC, Trichoderma, velo, cracker dry
- **FAQ** — Ciclos, contaminación, cómo cosechar, almacenaje

## 🎨 Diseño

- **Single-page app** en React (sin build, corre en cualquier navegador)
- **Paleta dual**: verde para cannabis 🌿, marrón para setas 🍄
- **Toggle de guía** en header — cambia colores y contenido al instante
- **Desktop sidebar** (900px+) — navegación vertical
- **Mobile horizontal** — scrolling de pestañas
- **Dark mode** por defecto — mejor para lectura nocturna

## 🚀 Cómo usar

1. **Abre en navegador**: `https://tecladooscuro.github.io/guia-cultivo/`
2. **Toggle guía** arriba a la izquierda: 🌿 Cannabis o 🍄 Setas
3. **Elige modo**: Interior/Exterior (cannabis) o Kit/Manual (setas)
4. **Navega tabs**: pestañas horizontales o sidebar vertical (desktop)
5. **Busca dentro**: Glosario tiene búsqueda integrada

**No requiere instalación ni cuenta. Funciona offline después del primer load.**
